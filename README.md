## 📌 Opis projektu
CloneTube to nowoczesny klon YouTube – responsywna aplikacja webowa do przeglądania, wyszukiwania i odtwarzania wideo.  
Zbudowana w oparciu o **Next.js (App Router)**, **TypeScript** i **Tailwind CSS**.  
Projekt skupia się na czystym, szybkim UI, server-side rendering, optymalizacji i dobrym UX – świetny przykład pełnowartościowego klonu popularnej platformy w nowoczesnym stacku.

## 🚀 Uruchamianie projektu
### 1️⃣ Klonowanie repozytorium
```bash
git clone https://github.com/michalpawlowski-mp/CloneTube.git
```
```bash
cd CloneTube
```

### 2️⃣ Instalacja zależności
```bash
 npm install  
```
lub  
```bash
yarn install  
```
lub  
```bash
pnpm install
```

### 3️⃣ Uruchomienie projektu
```bash
 npm run dev  
```
lub  
```bash
yarn dev  
```
lub  
```
pnpm dev
```

Aplikacja powinna być dostępna pod adresem:  
http://localhost:3000/  (standardowy port Next.js)
